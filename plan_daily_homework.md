Design a modern Learning Management System (LMS) Course Builder interface similar to platforms like Kajabi, Teachable or LearnDash.

The interface should include three main areas: course curriculum panel, lesson editor, and content block selector.

Use a clean professional dashboard UI, light background, subtle shadows, rounded cards, and accent color lime green for primary actions.
